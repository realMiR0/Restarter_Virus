1. Open the "Installer.py" file. 
2. type **Y** to accept the rules.
3. In main menu, choose **1 (Start)** to begin creating your app.
4. Pick one of the listed styles or choose **Custom** to set your own name and icon.
5. Wait until the build process finishes — your `.exe` file will be created in the **/dist** folder.
6. After “Finished!” message appears, type anything to **exit** the program.
